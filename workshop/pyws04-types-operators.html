PyWS04 - Types and Operators¶

Everything is an object. Every object has a type and an id.

Object Types in Python¶

  • None - has its own type, is one single object and has the boolean value of False
  • True, False - two boolean constants

Truth Value Testing¶

The following expressions are False:

  • None
  • False
  • Zero of any numeric type, for example, 0, 0.0, 0j.
  • Any empty sequence or mapping, for example, ‘’, (), [], {}.

Everything else is True

  • Boolean Expressions - and, or, not
  • Boolean Expressions - Comparisons (<, >, ==, !=, ...)
  • Boolean Expressions - Identity test (is, is not) should be used for test of None

Objects of different types never compare equal, except different numeric types. Comparisons of > or < work only for types, where a defined ordering of values exists.

  • Numeric Types - int, float, complex

  • Bitwise Operations on Integer Types (| (or), ^ (xor), & (and) <</>> (left/right shift), ~ (invert)

  • Iterator Types
    • Generators
  • Sequence Types — list, tuple, range
    • Immutable Sequence Types
      • Tuples
      • Ranges
    • Mutable Sequence Types
      • Lists

  • Text Sequence Type — str (‘abc’, “def”, ‘’‘ghi’‘’) - Strings are, by default, Unicode

  • Binary Sequence Types — bytes, bytearray, memoryview
    • Bytes

  • Set Types — set, frozenset

  • Mapping Types — dict

  • Context Manager Types

  • Modules

  • Classes and Class Instances

  • Functions
    • Methods

Operators and Operator Priority¶

Higher priorities start from top of the table.

Operator Description
() Parentheses (grouping)
f(args...) Function call
x[index:index] Slicing
x[index] Subscription (numeric or with key)
x.attribute Attribute reference (dot-operator)
** Exponentiation
~x Bitwise not
+x, -x Positive, negative
*, /, //, % Multiplication, division, remainder
+, - Addition, subtraction
<<, >> Bitwise shifts
& Bitwise AND
^ Bitwise XOR
| Bitwise OR
in, not in, is, is not, !=, ==, <, <=, >, >= Membership, identity, Comparisons
not x Boolean NOT
and Boolean AND
or Boolean OR
lambda Lambda expression
