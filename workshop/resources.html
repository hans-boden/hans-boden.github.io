Workshop Resources for Download¶

Show the names, which are defined¶

This is a little helper program, which can be executed in the IDLE. Then there are two functions available: ‘n()’ and ‘ni()’, which show a list of the names, which were defined.

Download this code and execute it in the IDLE. The code itself is not meant for beginners.

Show the usage of a graphical user interface¶

A very simple (text-based) user interface, indeed.

Make a new subdirectory and place these 3 download files there. Then try to run the calculator example. You will not need to touch the text_gui.py, but you can and should work with the calculator_gui.py and the mastermind_gui.py

Use a GUI to produce some Image output¶

Make a new subdirectory and place these 2 download files there. Save the gui4image_z.py - you can look at the code, but better not change it. If you want to experiment, make a copy. Then load the gui_sample.py - dont change this either (so you can come back and see how it works), but make a copy of it for your own drawing projects.
